러닝센터 셀프 스튜디오 이용 만족도 설문조사

러닝센터 셀프 스튜디오에서는
사용자가 직접 장비를 사용하고 녹화한 경험을 평가하여
개선사항을 파악하고 더 나은 서비스를 제공하고자 합니다.

소중한 의견 부탁드립니다.

⏱ 소요시간 약 1~3분 🔒 응답은 서비스 개선 목적으로만 사용됩니다
1

사용자

사용자의 소속을 적어주세요.*
예시 : Employee eXperience&Growth팀
러닝센터 셀프 스튜디오 사용 경험을 선택하여 주세요.*
2

스튜디오

장비 사용*
스튜디오 장비 사용에 어려움은 없으셨나요?
보완이 필요한 부분
장비 사용이 어렵다고 평가하신 경우, 어떤 점이 보완되면 좋을지 작성해주세요.
예시 : 장비의 사용 순서 표시 필요
이용가이드*
장비 사용에 대한 매뉴얼은 도움이 되셨나요?
보완이 필요한 부분
제공된 매뉴얼의 자료가 만족스럽지 않았다고 평가하신 경우, 어떤 보완이 이루어지면 좋을지 작성해주세요.
예시 : 매뉴얼의 이미지 확대 필요
3

리뷰

문제 해결
스튜디오 사용 중 문제가 발생했다면, 어떤 지원을 받았나요?
예시 : 헤드셋 고장으로 인한 교체 지원
보완이 필요한 점
문제 해결에 있어, 어떤 점이 더 보완되면 좋을지 적어주세요.
예시 : 주기적인 장비점검(장비체크)
향후 이용계획*
향후 러닝센터 셀프 스튜디오를 이용할 계획이 있으신가요?
기타 의견을 자유롭게 작성해주세요.
